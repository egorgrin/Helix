Форумы поддержки

Можно использовать редактор тем для внесения изменений в отдельные CSS- и PHP-файлы вашей темы.

Для начала выберите тему в выпадающем меню и нажмите кнопку «Выбрать». Появится список шаблонов темы. Нажмите на имя любого файла, чтобы загрузить его в редактор.

При редактировании PHP-файлов можно использовать выпадающее меню «Документация» для выбора функций, содержащихся в файле. Кнопка «Поиск» ведёт на веб-страницу с информацией об этой функции.

При использовании клавиатуры для навигации:

  • В области редактирования кнопка Tab вставляет символ табуляции.
  • Чтобы покинуть эту область, нажмие Esc и следом Tab.
  • При использовании программы чтения с экрана: в режиме форм вам потребуется нажать кнопку Esc дважды.

После внесения изменений нажмите «Обновить файл».

Совет: Если вы редактируете тему, используемую в настоящий момент, подумайте о возможном нарушении работоспособности сайта.

При обновлении темы сделанные вами изменения будут перезаписаны. Чтобы избежать этого, можно создать дочернюю тему.
